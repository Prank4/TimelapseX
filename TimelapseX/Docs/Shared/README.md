# iOS Dev Playbook

This repository contains reusable rules, principles, and workflows
used across all iOS projects.

It is designed to:
- Enforce architectural discipline
- Prevent AI-driven overengineering
- Keep MVPs small and shippable
- Capture learnings permanently

Each app project takes a **snapshot** of these documents.
This repo is the source of truth.
